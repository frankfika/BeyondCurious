========================================
杭州智领云帆科技有限公司 技术架构说明
========================================

一、整体架构

公司采用云原生微服务架构，支持公有云、私有云、混合云多模式部署，分为以下层次：

┌──────────────────────────────────────────┐
│              前端层 (Frontend)              │
│  React 18 / Vue 3 / 微前端(qiankun)        │
│  跨端：Electron / Flutter / 小程序         │
├──────────────────────────────────────────┤
│              网关层 (API Gateway)           │
│  Kong / Nginx / 认证鉴权 / 限流熔断 / 灰度发布 │
├──────────────────────────────────────────┤
│              服务层 (Microservices)         │
│  Java Spring Cloud / Python FastAPI / Go   │
│  用户服务 / 订单服务 / AI推理服务 / 数据服务    │
│  消息服务 / 权限服务 / 计费服务              │
├──────────────────────────────────────────┤
│              AI引擎层 (AI Engine)           │
│  PyTorch 2.0 / TensorFlow / ONNX Runtime   │
│  vLLM / TensorRT-LLM / LangChain           │
│  NLP引擎 / CV引擎 / 推荐引擎 / 知识图谱引擎   │
│  大模型推理引擎 / Agent编排引擎             │
├──────────────────────────────────────────┤
│              数据层 (Data Layer)            │
│  PostgreSQL 15 / MySQL 8 / Redis 7         │
│  Elasticsearch 8 / MinIO / ClickHouse      │
│  Apache Kafka / Apache Flink               │
├──────────────────────────────────────────┤
│              基础设施 (Infrastructure)       │
│  Kubernetes 1.28 / Docker / Prometheus     │
│  Grafana / ELK Stack / Jaeger              │
│  CI/CD: GitLab CI + ArgoCD                 │
│  部署：阿里云ACK + AWS EKS + 自建GPU集群     │
└──────────────────────────────────────────┘

二、核心技术栈

【前端技术】
- 框架：React 18、Vue 3、TypeScript 5
- UI组件：Ant Design Pro 6、Element Plus
- 状态管理：Redux Toolkit、Pinia
- 可视化：ECharts 5、D3.js、AntV G6
- 微前端：qiankun 2

【后端技术】
- 主语言：Java 17 (Spring Cloud Alibaba)、Python 3.11 (FastAPI)
- 辅助语言：Go (高性能服务)、Rust (核心算法)
- 服务治理：Nacos、Sentinel、Dubbo 3
- 消息队列：Apache Kafka 3.5、RabbitMQ
- 任务调度：XXL-Job、Apache Airflow

【AI技术栈】
- 深度学习框架：PyTorch 2.0、TensorFlow 2.x
- 大模型：Hugging Face Transformers、vLLM、TensorRT-LLM
- NLP：spaCy、jieba、NLTK
- CV：OpenCV、Detectron2、MMCV
- Agent框架：LangChain、AutoGPT
- 向量数据库：Milvus、Pinecone

【数据技术栈】
- 关系数据库：PostgreSQL 15、MySQL 8
- NoSQL：Redis 7、MongoDB 6
- 搜索引擎：Elasticsearch 8
- 数据仓库：ClickHouse、Apache Doris
- 数据湖：Apache Iceberg
- 实时计算：Apache Flink、Apache Kafka Streams
- 对象存储：MinIO、阿里云OSS

【基础设施】
- 容器编排：Kubernetes 1.28
- 服务网格：Istio 1.20
- 监控：Prometheus + Grafana + AlertManager
- 日志：ELK Stack (Elasticsearch + Logstash + Kibana)
- 链路追踪：Jaeger、SkyWalking
- CI/CD：GitLab CI + ArgoCD + Harbor
