# AI赋能投资研究：从效率工具到智能助手

> **面向群体**：一级市场投资人、PE/VC从业者、投研分析师
> **课程目标**：帮助投资人将重复性工作交给AI，聚焦高价值判断

## 课程简介

本课程专为一级市场投资人设计，系统讲解如何利用AI工具提升投研效率。从公域资料处理到私域知识库构建，从单点工具使用到多Agent协作工作流，帮助投资人建立完整的AI赋能投研体系。

## 课程模块

| 模块 | 主题 | 时长 | 核心内容 |
|-----|------|------|---------|
| 模块0 | 系统化文件整理 | 30分钟 | 批量分类、重命名、格式转换 |
| 模块1 | NotebookLM实战 | 45分钟 | 公域资料处理、知识库搭建 |
| 模块2 | 本地化RAG系统 | 90分钟 | RAGFlow部署、私域知识库构建 |
| 模块3 | 多Agent协作 | 90分钟 | 研报撰写工作流、分章节并行 |
| 模块4 | 最终输出与交付 | 45分钟 | PDF生成、PPT制作 |

## 核心工具栈

- **NotebookLM** - 公域资料智能问答
- **RAGFlow** - 私有化知识库部署
- **Claude Code** - 代码自动化与工作流
- **Obsidian** - 笔记整理与知识沉淀
