## Prompt 示例库

### 示例1：唐明皇和杨玉环在时代广场

```
请创作一张极其随意、毫无构图考量的自拍照，呈现出一种普通手机快照的感觉。

**主角为杨贵妃和唐明皇**，身穿华丽的唐式服装，但姿态放松随意。
拍摄地点位于**美国时代广场**，霓虹灯牌、广告大屏和街头LED的炫光映在他们脸上。
时间是夜晚，手机镜头因强光产生眩光。

整体氛围要像穿越剧穿帮现场——盛唐宫廷人物误入现代纽约，画面充满违和感和无厘头的欢乐。
```

### 示例2：元气少女自拍

```
**提示词：可爱少女的元气日常**

**画面风格**：胶片质感，富士胶卷色调，低颗粒，边缘自然漏光
**人物设定**：20岁左右亚洲少女，空气刘海+微卷马尾，淡妆
**服装**：宽松浅色卫衣+牛仔背带裤，彩色帆布鞋
**姿势**：坐在公园长椅上，双手捧着奶茶，对镜头眨眼微笑
**环境**：阳光下的公园，虚化的绿树和散步的人群
**光线**：午后阳光，在头发和奶茶杯上形成柔和光晕

**氛围关键词**：清新自然/日常治愈/阳光少女的简单快乐
```

### 示例3：水墨画小老虎

水墨画风格，一只拟人化黑色的胖胖的毛茸茸的可爱的小老虎，

## 使用技巧

### 1. 结构化描述

```
【主体】+【动作】+【环境】+【光线】+【风格】
```

### 2. 即梦AI 模型选择

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| 图片3.0 | 影视质感，文字嵌入好 | 高质量图片 |
| 图片2.1 | 平面绘制感 | 海报设计 |
| 图片2.0Pro | 创意性强 | 写真摄影 |
| 图片2.0 | 文字遵循度高 | 带文字图片 |

### 3. Gemini 图片修改

Gemini 支持通过自然语言修改图片：

```
请将这个小女孩的头发变成白色的
我希望这个小女孩正在站起来投篮
把衣服上的阿迪达斯改成"Frank"
```

## 高级技巧

### 负面提示词

```
不要：模糊、低质量、变形、水印、文字重叠
```

### 风格修饰符

```
--ar 16:9       # 宽高比
--style vivid   # 风格
--quality 2     # 质量
```

## 注意事项

1. 描述越具体，效果越好
2. 使用中文模型时，中文 Prompt 效果更好
3. 复杂场景可能需要多次尝试
4. 注意版权问题，不要用于商业用途
