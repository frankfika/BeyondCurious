## 框架定义

ICIO 框架是一种清晰的四要素结构，特别适用于数据处理和分析任务。

| 要素 | 英文 | 中文 | 说明 |
|------|------|------|------|
| I | Instruction | 指令 | 明确具体的任务要求 |
| C | Context | 语境 | 提供必要的背景信息 |
| I | Input Data | 输入数据 | 待处理的具体信息 |
| O | Output Indicator | 输出指示器 | 定义处理结果的风格和格式 |

## 应用示例：雅思老师讲解复杂句

```markdown
# Role
你是一位专业的雅思老师，教学风格风趣幽默，善于通过中英文对比的方法帮助学生理解英语知识点。

## Instruction（指令）
请用风趣幽默的语言，结合中英文对比，讲解雅思写作中"复杂句"的用法和注意事项。

## Context（语境）
学生正在准备雅思写作，希望通过复杂句来提高写作分数。

## Input Data（输入数据）
- 复杂句定义
- 常见复杂句结构（定语从句、状语从句、名词性从句等）
- 中英文对比例句

## Output Indicator（输出指示器）
- 语言风趣幽默
- 结合中英文对比，通俗易懂
- 内容条理清晰，重点突出
```

## 使用步骤

1. **定义任务**：明确要完成的任务
2. **提供语境**：说明任务背景
3. **准备数据**：整理输入信息
4. **定义输出**：明确输出格式

## 优势

- 结构清晰，易于理解
- 适合数据处理任务
- 输出格式可控

## 注意事项

- Instruction 必须清晰明确
- Context 提供充分但不冗余
- Input Data 要具体相关
- Output Indicator 包含格式、风格要求
