## 核心原理

链式思考（CoT）通过让 AI 展示推理过程，获得更准确的答案。

## 三大核心步骤

1. **问题分解**：将复杂任务拆解为子步骤
2. **提供范例**：提供标准推理模板
3. **强制推理**：要求展示中间步骤

## 应用示例：《红楼梦》人物关系分析

```markdown
请按照链式思考的方法分析《红楼梦》中的人物关系：

第一步，识别并列出小说中的主要人物；
第二步，梳理人物之间的血缘关系、社会关系和利益冲突；
第三步，结合贾府衰落的历史背景，分析人物命运的必然性。

请逐步展示你的推理过程，不要直接给出最终答案。
```

## 通用 CoT 模板

```markdown
请按以下步骤解决问题：

步骤1：[明确第一步]
步骤2：[明确第二步]
步骤3：[明确第三步]

请逐步展示推理过程。
```

## 效果对比

### 直接提问（错误率高）
```
水池有进水管和排水管，单开进水管12小时注满，单开排水管24小时排空。
两管同开，几小时注满水池？
```

### 链式思考（准确率高）
```
请按以下步骤：
首先计算进水管每小时效率为1/12池；
然后计算排水管每小时效率为-1/24池；
接着求净效率：(1/12) + (-1/24) = 1/24池；
最后计算时间：1 ÷ (1/24) = 24h。
```

## 适用场景

- 数学问题
- 逻辑推理
- 文本分析
- 编程问题
- 决策分析

## 注意事项

1. 步骤要清晰有逻辑
2. 引导要明确
3. 简单问题无需 CoT
4. 现代"思考模式"模型已内置 CoT 能力
