## 说明

此 Prompt 用于测试大模型生成小红书风格旅游攻略的能力。评估大模型对社交媒体平台文案风格的理解和创作能力。

## Prompt 内容

你是一位专业的旅游博主和小红书写手，春节期间去青岛旅游了5天，请就此写一篇小红书风格的旅游攻略。

## 输出要求

一篇符合小红书风格的旅游攻略，应包含：

1. **吸引眼球的标题**：使用 emoji 和夸张的表达
2. **详细的行程安排**：每日行程、景点推荐
3. **美食推荐**：当地特色菜品、餐厅推荐
4. **实用 Tips**：穿搭建议、交通攻略、避坑提醒
5. **互动引导**：鼓励读者评论和关注
6. **相关标签**：#青岛旅游 #春节旅行 等

## 风格特点

小红书文案的典型特点：

- 使用大量 emoji 表情
- 短句为主，段落简洁
- 口语化、接地气的表达
- 注重实用性和分享价值
- 带有个人体验和主观评价

## 使用方法

1. 复制上述 Prompt
2. 粘贴到大模型对话框
3. 查看生成的小红书风格旅游攻略
4. 根据需要调整细节（如特定景点、美食偏好）

## 模型表现参考

根据书中测试：

- **Qwen3**：表现最佳，符合小红书风格，emoji 使用恰当
- **DeepSeek**：内容丰富，有本地人认证的实用价值
- **Claude**：结构清晰，实用性较强
- **GPT-4o**：内容完整，但风格略显传统
- **Grok3**：符合要求，但 emoji 使用较少

Day1-2：老城暴走·胶片感年味

1️⃣ 船歌鱼水饺（鲅鱼水饺鲜到掉眉毛）
