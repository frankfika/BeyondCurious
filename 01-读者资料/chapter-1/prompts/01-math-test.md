## 说明

此 Prompt 用于测试大模型的数学逻辑推理能力。通过一道关于整除的数学题目，评估大模型的逻辑推理和问题解决能力。

## Prompt 内容

你是一位数学专家，请帮忙解答如下题目，并给出结果和推理过程：已知正整数 n 满足 n²+1 能被 n+1 整除，求所有满足条件的正整数 n。

## 预期输出

正确的解答过程：

1. 使用多项式除法或代数恒等式分析问题
2. 推导出 n+1 必须整除 2
3. 得出 n = 1 的结论
4. 验证答案的正确性

## 使用方法

1. 复制上述 Prompt
2. 粘贴到大模型对话框（DeepSeek、ChatGPT、Claude 等）
3. 查看模型的解答过程和结果
4. 评估推理逻辑的正确性

## 测试结果参考

根据书中测试，各模型都能正确解答此题：
- GPT-4o：正确，n = 1
- Qwen3-256B：正确，n = 1
- Grok3：正确，n = 1
- Claude 4-sonnet：正确，n = 1
- DeepSeek 671B：正确，n = 1

## 正确解答

```
解题过程：
n² + 1 = (n + 1)(n - 1) + 2

因此：
(n² + 1) ÷ (n + 1) = (n - 1) + 2/(n + 1)

要使结果为整数，2/(n + 1) 必须为整数
即 n + 1 必须整除 2

由于 n 是正整数，n + 1 ≥ 2
2 的正因数只有 1 和 2

n + 1 = 1 → n = 0（不符合正整数要求）
n + 1 = 2 → n = 1

验证：当 n = 1 时，(1² + 1) ÷ (1 + 1) = 2 ÷ 2 = 1（整数）

答案：n = 1
```
