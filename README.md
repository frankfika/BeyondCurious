# BeyondCurious

对外分享 AI 内容的资料库：读者资料、课程专辑、文章笔记、工具箱与可运行示例。

如果你是第一次来这里，建议从 [START-HERE.txt](./START-HERE.txt) 开始。

## 目录结构

这个仓库按“文件柜”方式组织：每个顶层文件夹就是一个抽屉。

- `01-读者资料/`：书籍读者会用到的配套资料、勘误、示例与下载内容
- `02-课程与专辑/`：成体系教程内容（课程讲义、练习、作业、配套材料等）
- `03-文章与笔记/`：文章类内容与日常积累（对外分享/对内复盘）
- `04-AI工具箱/`：让 AI 更好用的工具与配置（MCP、Skills、提示词与工作流等）
- `05-示例项目/`：可运行的最小示例与演示项目（便于照着跑）
- `06-图片与课件/`：会被反复引用的素材（图片、图表、课件、下载物等）
- `99-自动化/`：脚本与自动化工具（可选）

规范与约定见 [catalog.yml](./catalog.yml)。

## 使用方式

- 直接浏览：按目录进入各主题内容
- 搜索定位：用 GitHub 搜索关键字（标题/小节/代码片段）
- 复现示例：进入 `05-示例项目/` 按每个示例的说明运行
