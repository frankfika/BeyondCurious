## 项目简介

BeyondCurious 是一个结构化的 AI 知识管理平台，作为《动手做 AI Agent：零基础玩转智能体》书籍的配套资料库，同时也是一个综合性的 AI 学习资源中心。

本项目采用 Obsidian 进行内容管理，通过清晰的目录结构组织 AI Agent、提示词工程、AI 工具等核心内容，便于读者系统学习和实践。

## 🌟 核心内容

### 1. 📚 读者资料 (01-读者资料/)
书籍配套材料，按章节组织：
- **第一章**：AI Agent 基础概念与环境搭建
- **第二章**：进阶提示词工程（ICIO、RISE、CRISPE、BROKE 框架）
- **第三章**：实战项目与案例解析

### 2. 🎓 课程与专辑 (02-课程与专辑/)
系统化教程内容：
- 完整课程讲义与练习
- 专题系列文章
- 配套作业与解答

### 3. 📝 文章与笔记 (03-文章与笔记/)
- 技术文章精选
- 日常学习笔记
- 实践经验分享

### 4. 🛠️ AI 工具箱 (04-AI工具箱/)
- **MCP 插件**：Model Context Protocol 配置与示例
- **Claude Skills**：自定义技能定义
- **提示词模板**：经过验证的高效 Prompt
- **工作流程**：自动化工作流配置

### 5. 🚀 示例项目 (05-示例项目/)
可运行的最小示例：
- MetaGPT 软件公司 Demo
- Flask AI API 服务
- N8N 工作流示例

### 6. 🖼️ 图片与课件 (06-图片与课件/)
可复用媒体资源：
- 技术架构图
- 课程 PPT
- 信息图表
